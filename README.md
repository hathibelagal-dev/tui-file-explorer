- Written in pure C, so it's extremely fast and lightweight.

### ⌨️ Controls

**w** - go up a file

**s** - go down a file

**space** - select current directory

**p** - go to parent directory

**b** - go back to previous directory

**r** - redraw (necessary after resizing the terminal window)

**q** - quit
